### 졸음운전 예방 및 능동제어 시스템
* **설명**: 복합 센서 융합(Sensor Fusion)을 적용한 Fail-safe 졸음 방지 및 운전 보조 시스템
* **기간**: 8일, 5인 팀 프로젝트
* **핵심 역할**: 팀장, 전체 SW 아키텍처 및 통신 인터페이스(ICD) 설계 주도, 복합 센서 기반 졸음 위험도 판단 및 통신 라우팅을 담당하는 게이트웨이 노드 개발, 모니터링 대시보드 구현
* **성과**: 부품 누락 및 MCU 보드 변경 이슈 발생 시 통신 인터페이스를 유연하게 전환하고 더미 데이터를 활용하여 전체 시스템을 시나리오 결함 없이 검증 완료, 위험도 임계값 튜닝을 통해 오경보와 미감지 간의 균형점 도출
* 🔗 [Link to Project](https://github.com/sesac-mobility-embedded-project-team2/highway-drowsiness-prevention)

### 카카오맵 장소 검색 및 엑셀 변환 프로그램
* **설명**: 카카오맵 API와 우체국 우편번호 API를 연동하여 특정 키워드의 장소 정보를 검색하고, 그 결과를 엑셀 파일로 일괄 저장하는 Java 기반 데스크톱 애플리케이션
* **기간**: 2주, 개인 프로젝트
* **핵심 역할**: 전체 설계 및 개발 주도(1인), Java Swing을 활용한 사용자 친화적 GUI 구현, REST API 통신 및 JSON/XML 응답 데이터 파싱 처리, Apache POI 라이브러리를 활용한 엑셀 파일(.xlsx) 자동 생성 로직 개발, Dotenv를 이용한 환경 변수(API Key) 분리 및 파일 기반 오류 로깅 시스템 구축
* **성과**: 수작업으로 진행되던 장소 주소 및 우편번호 수집 과정을 자동화하여 데이터 확보 및 문서화 효율성 증대, 서로 다른 두 개의 외부 API(카카오맵 로컬 API, 우체국 오픈 API)를 안정적으로 연계하여 하나의 프로세스로 통합 완료
* 🔗 [Link to Project](https://github.com/yonguk98/kakaomaptoexcel)

### 성수동 팝업 스토어 큐레이션 및 지도 서비스
* **설명**: 인스타그램 자동 크롤링과 OCR 기술을 결합하여 팝업 스토어 정보를 수집하고, 지도 기반 큐레이션 및 후기 기능을 제공하는 풀스택 웹 서비스
* **기간**: 4개월, 1인
* **핵심 역할**: Selenium과 Quartz를 활용한 크롤링 및 스케줄링 파이프라인 구축, 네이버 Clova OCR 및 Map API 연동, Spring Boot 백엔드 API와 Vue.js 프론트엔드 UI/UX 전체 개발
* **성과**: 수작업에 의존하던 정보 수집 프로세스를 자동화하여 데이터 확보 효율성 증대, GitHub Actions와 Docker(GraalVM)를 이용한 CI/CD 파이프라인 구축 완료, JWT 기반 인증 및 반응형 지도 인터페이스 구현으로 사용자 경험 개선
* 🔗 [Backend Repository](https://github.com/yonguk98/popup_capstone) | 🔗 [Frontend Repository](https://github.com/yonguk98/popup_front_capstone)
